# AMD-education-data-analysis


This project demonstrates an end-to-end data pipeline and interactive dashboard for analyzing ophthalmic patient education modules.

## Features
- ETL pipeline with SQLite integration
- Demographic and quiz score analysis
- Interactive dashboard built with Streamlit
- Word cloud and sentiment analysis for patient feedback

## Tech Stack
Python, SQLite, Streamlit, Plotly, Pandas, Matplotlib, WordCloud

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run the dashboard: `streamlit run streamlit_dashboard.py`

## Data Disclaimer
Synthetic data is used for demonstration purposes. Real patient data is excluded
